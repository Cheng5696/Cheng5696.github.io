FDM基础_耗材

发表于 2024-08-23 | 分类于 3D打印

FDM的材料要求

成型材料要求

由以上材料特性对FDM工艺实施的影响来看，FDM工艺对成型材料的要求是熔融温度低，粘度低，粘结性好，收缩率小。

  1. 材料的粘度
    材料的粘度低、流动性好，阻力就小，有助于材料顺利挤出。材料的流动性差，需要很大的送丝压力才能挤出，会增加喷头的启停响应时间，从而影响成型精度。
  2. 材料熔融温度
    熔融温度低可以使材料在较低温度下挤出，有利于提高喷头和整个机械系统的寿命。可以减少材料在挤出前后的温差，减少热应力，从而提高原型的精度。
  3. 材料的粘结型
    FDM工艺是基于分层制造的一种工艺，层与层之间往往是零件强度最薄弱的地方，粘结性的好坏，决定了零件成型以后的强度。粘结性过低，有时在成型过程中因热应力会造成层与层之间的开裂。
  4. 材料的收缩率
    由于挤出时，喷头内部需要保持一定的压力才能将材料顺利挤出，挤出材料丝一般会发生一定程度的膨胀。如果材料收缩率对压力比较敏感，会造成喷头挤出的材料丝直径与喷嘴的名义直径相差太大，影响材料的成型精度。FDM成型材料的收缩率对温度不能太敏感否则会产生零件翘曲、开裂。

支撑材料的要求

  1. 能承受一定高温度
    支撑材料必须能承受成型材料的高温
  2. 与成型材料不浸润
  3. 具有水溶性或者酸溶性
  4. 具有较低的熔融温度
    提高喷头使用寿命
  5. 流动性要好
    支撑材料的成型精度要求不高，为了提高机器的打印速度，要求支撑材料具有很好的流动性，相对而言，毫于粘性可以差一些。

耗材特点

常规耗材

PLA

  1. 容易打印

  2. 颜色丰富

  3. 对机器要求低

  4. 模型表面质量好

  5. 不耐老化3-5年会变脆

  6. 不防紫外线避免晒太阳

  7. 不耐水/酸碱

  8. 韧性差

不适合打结构性零件，只适合打摆件装饰品。

一般打印温度190-200°、热床40-50°

PETG

  1. 韧性好

  2. 耐老化/紫外线

  3. 耐水/酸碱

  4. 相比其他工程材料好打印

  5. 不如PLA好打印

适合打结构件，实用性模型

PETG是PET的改性材料，PET一般用来做餐具，饮料瓶。

PETG线材有较强的吸水性，裸露在空气中更容易变质。

TPU(柔性耗材)

一般打印温度210-220°、热床30-40°

根据硬度做选择 例如85A，90A,95A 数字越小越软

  1. 要求近程挤出打印机
  2. 打印速度慢
  3. 容易堵头
  4. 不适合打印过高的模型

工程塑料

ABS

一般打印温度230-250°、热床80-100°

  1. 打印大面积非常容易翘边

  2. 打印时会发出有毒气体，(例如苯)，需要在通风条件好的情况下打印。

  4. 模型结实耐摔，耐高温，耐水。

PC(聚碳酸酯)

  1. 耐热性：PC 耗材具有良好的耐高温性能，可在相对较高的温度下保持稳定性。
  2. 耐冲击性：PC 耗材具有出色的抗冲击性，适合用于制造需要抗冲击保护的零部件。
  3. 透明性：PC 耗材具有良好的透明度，可用于制作透明或半透明的零部件和模型。
  4. 耐化学性：PC 耗材对许多化学品具有较好的耐受性，适用于一些特殊环境下的应用。

PC 耗材适合用于制造要求高强度、高温稳定性和透明度的部件和模型。

Asa

  1. 户外性能好，可以长期暴漏在强紫外线光下。

PA(尼龙Nylon)

一般打印温度210-220°、热床50-60°

  1. 耐磨性：尼龙具有出色的耐磨性，适合制造需要耐磨的零部件和机械组件。
  2. 耐高温性：尼龙耗材具有良好的耐高温性能，可在相对高温下保持稳定性。
  3. 高强度：尼龙具有较高的强度和刚度，适合制造要求高强度的零部件和结构。
  4. 耐化学性：尼龙耗材对许多化学品具有良好的耐受性，适合在挑战性环境中使用。

尼龙耗材常用于制造轴承、齿轮、连接件、工装夹具等要求高耐磨性和高强度的部件。

高温耗材

PEEK

用途

  1. 制造骨科植入物，如椎间盘植入物、骨板和螺钉等，因其生物相容性和耐腐蚀性而受到青睐。
  2. 用于制造航空发动机零部件，如密封圈、阀门座圈等，其高温耐受性和机械性能能够满足航空发动机的高要求
  3. 用于汽车发动机的零部件制造，如密封圈、节温器、液压阀等
  4. PEEK也可用于制造汽车传动系统的零部件，如齿轮、轴承座等

打印温度
一般打印温度通常设置在 360°C - 400°C 之间

打印时需要注意的事项

  1. PEEK对温度波动敏感，因此建议使用具有加热室的3D打印机
  2. 具备高温耐受能力的3D打印机和热端组件
  3. PEEK 3D打印通常需要较高的热床温度以确保打印物件的附着性。建议将热床温度设置在 120°C - 150°C 之间
  4. PEEK对湿气敏感，建议在打印前将PEEK材料存放在干燥的环境中，以避免影响打印质量。

改性耗材

-CF(Carbon fiber 碳纤维)

碳纤维
碳纤维并不是一种传统意义上的3D打印耗材，而是一种用于制造复合材料的增强材料。在3D打印中，通常不直接使用碳纤维作为单独的耗材，而是将碳纤维与其他塑料材料结合制成碳纤维增强复合材料。这种复合材料通常具有优越的强度、刚度和耐热性。

碳纤维增强复合材料的特点：
高强度，高刚度，轻质化

加了碳纤维的耗材不建议使用黄铜喷嘴打印

  1. 碳纤维的硬度可能导致对黄铜喷嘴的较快磨损，特别是在长时间打印或使用高速打印时更容易受到影响。
  2. 可能会导致 PLA-CF 更容易在黄铜喷嘴内堵塞，尤其是在细孔喷嘴中更为明显。

PLA-CF PETG-CF 尼龙-CF
在原材料中添加了碳纤维粉末。

  1. 强度提升
  2. 模型表面细磨砂质感

PLA+ / PETG+

韧性提升
不提升打印难度

耗材干燥

PLA干燥

50度 5-6个小时

PETG干燥

60度 4-6个小时
